# Reproducing Simulations of AI-assisted Survey Data Collection

This repository contains code to reproduce the simulation study.

## Data

The Dutch General Population Survey (GPS) data can be downloaded from the companion website of the *Handbook of Nonresponse in Household Surveys*: <https://survey-nonresponse.com/> . This repository does not redistribute the GPS data.

## Project structure

-   `scripts/`: analysis scripts
-   `paper/`: manuscript source
-   `output/figures/`: generated figures
-   `output/tables/`: generated tables
